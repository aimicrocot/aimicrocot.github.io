# Сайт-визитка

Минималистичный сайт-визитка с набором полезных материалов: расширениями для SillyTavern, гайдом по промптингу в РП и готовым пресетом для гибкой, живой и экономной для токенов РП.

## Демо

Открыть проект: https://aimicrocot.github.io/

## О проекте

Этот сайт создан как удобная точка входа, где собраны необходимые инструменты и инструкции в одном месте. Подходит для быстрого старта и упрощения задач а РП.

## Для кого

- Опытные в РП
- Новички
- Пользователи, которым нужен готовый набор инструментов и настроек

## Содержимое

### Расширения
Подборка полезных расширений для удобства во время РП.
- Счётчик токенов: всех потраченных вами
- Сценарии: один бот, много сценариев
- Саммари: скрытие сообщений для экономии, в разработке

### Гайд
Пошаговая инструкция по промптингу в SillyTavern.

### Пресет
Готовые промпты, написанные по руководствам OpenAI, Google и Anthropic..

## Использование

1. Перейдите на сайт
2. Ознакомьтесь с доступными материалами
3. Следуйте гайду
4. Загрузите пресет/расширения
